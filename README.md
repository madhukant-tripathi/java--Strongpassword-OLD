# Strong Password Checker (Java)

# Description
This Java program checks whether a password entered by the user is strong or weak
based on given rules.
It validates the password by checking its length, uppercase letters, lowercase
letters, digits, and special characters.
The password is accepted only when all conditions are satisfied.
This program was created to understand password validation logic using basic
Java programming concepts.

# Password Rules
- Minimum 12 characters
- At least 4 uppercase letters
- At least 4 lowercase letters
- At least 3 digits
- At least 1 special character

# Concepts Used
- Scanner class
- for loop and while loop
- if-else conditions
- Character comparison

# How to Run
1. Compile the program
   javac Strongpassword.java
2. Run the program
   java Strongpassword

# Output
- Displays messages if the password is weak
- Accepts the password when all rules are satisfied
